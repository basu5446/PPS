Dear Students , All have to create a project based on your learning in “PPS using C” course. Guidelines :: 1. Team Size – 4 members 2. Level of project – K4 (Suggested Projects are attached) 3. Work of each team member should be justified. 4. Each project should be unique. THANK YOU SITUDENTS
  • Topic 1: Building a Calendar using C.
  • Topic 2: Build a Number system conversion from any base to any base conversion.
  • Topic 3: BMI Calculator using C BMI is calculated by dividing a person's weight in kilograms by the square of their height in meters (BMI = weight(kg) / height(m)²).
  • Topic 4: Student record management using C
  • Topic 5: Tic-Tac-Toe by Text. Build a Tic-Tac-Toe game that’s playable like a text adventure. Can you make it print a text-based representation of the board after each move?
  • Topic 6: Simple Food Ordering System using C
  • Topic 7: Bank Account Management System: Write a C program to simulate a simple bank account system where a user can perform operations such as deposit, withdraw, and check the account balance. The system should allow users to: 1. Deposit money. 2. Withdraw money (only if the balance is sufficient). 3. Check the balance. You should create a menu-driven program that runs continuously until the user decides to exit.
  • Topic 8: Build a number-guessing game. Generate a random number using the random module. Prompt the user to guess a number within the specified range. Compare the user’s guess with the number If the guess is low or high, print “guess higher or low”. If guess matches the number, print “Congratulations! You guessed it right!!” and exit the game.
  • Topic 9: Build a Mini voting system using c.
  • Topic 10: Rock, Paper, and Scissor It is a very simple C project based on a popular game played by children called Rock, Paper, and Scissor. Here, the first user moves, and then the program makes one. You can use a single alphabet or input an entire string to indicate the move. You need to set up a function to check the move’s validity
